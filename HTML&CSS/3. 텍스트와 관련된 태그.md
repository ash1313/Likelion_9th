# 텍스트와 관련된 태그

### 텍스트와 관련된 태그

##### 제목 태그: 제목을 나타내고 싶을 때 사용, 중요도에 따라 1~6까지 씀

+ h1: 중요
+ h6: 덜 중요

##### 본문 태그

+ p태그-paragraphs: 단락, 문단
+ br태그-break: 종료태그를 쓰지 않는다(빈 요소), 엔터(줄바꿈)
+ pre태그-preformatted: 형식화된, 적은 그대로 화면에 출력

##### 글자와 관련된 태그 

+ strong태그-strong: 강조
+ em태그-emphasized : 기울기
+ sub태그-subscripted: 글자를 아래로 내림
+ sup태그-superscripted: 글자를 위로 올림
+ ins태그-inserted: 밑줄
+ del태그-deleted: 취소선

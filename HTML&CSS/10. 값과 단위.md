# 값, 단위, 색

### 숫자값과 백분율

+ 숫자값

  + (px->절대길이)

  + (em,rem->상대길이)

    + em: 현재 스타일이 지정된 요소의 font-size기준
    + rem: 최상위 요소의 font-size기준
    + 1em(rem)의 크기= 기준 font-size * 1em(1rem)
    + %(퍼센트)=상대 길이, 보통 이미지나 레이아웃의 너비나 높이를 지정할 때 씀

    

+ 색상

  + RGB
  + hex code
  + hsl

  

+ position: positon 프로퍼티에 사용할 수 있는 값
  + static 
  + absolute
  + relative
  + fixed
